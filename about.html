    • 你好，我是無限

    我第一次接觸程式是在大學，當時學的是 C 語言跟 Arduino，但其實那時候並不知道可以做什麼。

    後來我在營造業工作，為了幫主管計算線性插值，我去寫了一段 VBA，把原本要花一週做的工作，在兩小時內完成。那一刻我才真正體會到『程式可以改變工作的樣子』，我也因此決定轉職，報名了新尖兵的系統課程，走上軟體開發這條路。」。

    目前是一位以 JavaScript / Node.js 為核心的後端工程師，熟悉 Express 架構與 MongoDB 資料庫。

    過去兩年，我曾接手魔改後的 Pomelo 框架，獨立設計並開發四款即時多人連線遊戲的後端系統。實作內容涵蓋 WebSocket 心跳包、JWT 驗證、遊戲邏輯與帳務拆分、玩家上下線管理，以及與前端協作完成多人房間流程與畫面串接等。
